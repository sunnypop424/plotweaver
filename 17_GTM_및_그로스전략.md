# GTM 및 그로스 전략

> 작성: 그로스 리드 / 최종수정: 2026-06-29

> 플롯위버(PlotWeaver)의 시장 진입(Go-To-Market)과 성장 루프 설계.
> **북극성 지표(NSM) = MAU.** `08` 시뮬레이션상 매출 민감도는 **MAU > 구독전환율 > 단가**이며, **흑자는 MAU 3만부터**. 따라서 모든 그로스 활동의 1순위 목표는 "값싸게 MAU를 모으고, 바이럴로 CAC를 깎는다".
> 핵심 후크: **"글 못 써도, 내 설정이 진짜 소설 + 표지가 된다."**

## 1. 📌 포지셔닝 & 핵심 메시지

### 1-1. 포지셔닝 문장
> **"웹소설을 '쓰는' 도구가 아니라, '완성시켜주는' 스튜디오."**
> 경쟁(노벨AI류 생성기, 일반 챗봇)은 "AI가 글을 써준다"에 그치지만, 플롯위버는 **설정→회차 본문→표지→공유·수익화**까지 원스톱. 차별점은 **관계도 기반 일관성**과 **시각적 완성물(표지)**.

### 1-2. 퍼소나별 가치제안 한 줄 (메시지 후크)

| 퍼소나 | 한 줄 가치제안(후크) | 강조 베네핏 |
| --- | --- | --- |
| **A. 설정덕후 입문 창작자 (10대)** | **"글 못 써도, 네 설정이 진짜 소설이 되고 표지까지 나와."** | 자랑·완성감·표지 |
| **B. 연재 부담 현직 작가 (20·30대)** | **"마감 전날, 회차 초안을 30분 만에. 설정은 안 무너진다."** | 생산성·일관성·초안 속도 |
| **C. 취미·힐링 소비 독자 (30·40대)** | **"못 쓰는 게 아니라 안 쓴 것뿐. 30초 입력이면 내 취향 소설이 읽힌다."** | 최소 입력·즉시 결과 |

> 메인 카피(전 채널 공통): **"내가 만든 설정, AI가 소설로. 표지까지 공짜로 1편."**
> 서브 카피(전환용): "글재주 0, 끈기 0이어도 완결까지."

### 1-3. 메시지 위계
- **1차 후크(유입)**: 표지·완성작 비주얼 → "이거 AI로 만든 거야?"
- **2차 메시지(가입)**: "너도 1편 무료로 만들 수 있어"
- **3차 메시지(전환)**: "다음 화도 이어쓰고, 판매·후원까지"

## 3. 🔁 바이럴 루프 설계 (K-factor)

### 3-1. 핵심 루프 — "표지·완성작 공유 루프"
```
[표지·완성작 생성(Aha)] → [공유(SNS/디스코드/링크)] → [신규 노출]
   → [랜딩 유입] → [가입] → [무료 1편 생성(Aha)] → [다시 공유]  ↺
```
> 바이럴 트리거는 **AI 책 표지**(`01`). 글보다 이미지가 SNS 전파력이 압도적이므로, **표지를 공유 자산의 1순위**로 설계.

### 3-2. 공유 시점·후크 (제품에 내장)
| 공유 지점 | 장치 | 후크 카피 |
| --- | --- | --- |
| 표지 생성 직후 | **원클릭 공유 카드**(표지+제목+한줄소개) | "이거 나 AI로 만듦 ㅋㅋ" |
| 1회차 완성 직후 | 공유 링크 + 미리보기 카드 | "내 설정으로 쓴 소설, 1화 보고 가" |
| 무료 표지 워터마크 | **"made with PlotWeaver" 워터마크 + 추천코드** | 무료(Free) 플랜은 워터마크 강제 → 자연 노출 |
| 공유 랜딩 | 추천코드 자동 삽입 + "너도 무료 1편" CTA | 가입 시 양쪽 크레딧 보상 |

> Free 표지의 워터마크는 비용이 아니라 **무료 미디어 채널**이다(`05` Free=워터마크). 단, 자랑 가치를 해치지 않도록 디자인 비중을 작게.

### 3-3. K-factor 관리
- **K = (사용자당 평균 초대/공유 수 i) × (공유당 전환율 c)**
- 목표: 초기 **K ≥ 0.3~0.5**(완전 바이럴 K≥1은 비현실적, 유료+바이럴 병행)
- **i 올리기**: 공유 마찰 제거(원클릭 카드), 공유 인센티브(추천 크레딧), 챌린지로 공유 동기 부여
- **c 올리기**: 공유물 자체가 광고(표지가 매력적) + 랜딩 전환율 최적화(무료 1편 즉시 체험)
- **측정**: 공유 발생률, 공유→방문률, 방문→가입률, 추천코드 사용 가입 비중 (퍼널은 `15` 연계)

> 바이럴 보조 지표: **CAC를 K-factor가 얼마나 깎는가**. 유료 CAC × (1−바이럴 기여분) = 실효 CAC로 관리.

## 4. 🚀 런칭 플랜 (3단계)

### 4-1. 단계 개요
```
[Pre-Launch] 웨이트리스트   →   [Beta] 클로즈드→오픈   →   [GA] 정식 런칭
   가짜문(09 실험1)              제품·전환 검증              스케일·유료 가속
```

### 4-2. Phase 0 — 프리런칭(웨이트리스트)
- **목표**: 수요 검증 + 초기 창작자 풀 확보. `09` 실험1(랜딩+가짜문)과 **동일 자산 재활용**.
- **전술**: 랜딩 1장(표지·예시 노출) → 이메일 등록 CTA. 소액 광고(50만 원)로 타겟 유입. 대기열 순번·추천 시 순번 상승(바이럴 웨이트리스트).
- **KPI**: 랜딩→등록 전환율 **≥ 8%**, CPL **≤ 3,000원**(`09` 성공 기준 그대로), 웨이트리스트 **1,000명+**.
- **채널**: 디스코드/X 작가 커뮤니티/숏폼 티저.

### 4-3. Phase 1 — 클로즈드 베타
- **목표**: Aha 도달률·원가·전환율 검증(`09` 실험2·3과 동행).
- **전술**: 웨이트리스트 + **시드 창작자 50~100명** 초대(공급 먼저). 무료 크레딧 넉넉히. 1:1 피드백 채널(디스코드).
- **KPI**: **활성화율(가입→1회차+표지 완성) ≥ 40%**, D7 리텐션 ≥ 20%, 생성 1건 원가<판매가 신호 확인.

### 4-4. Phase 2 — 오픈 베타
- **목표**: 바이럴 루프 가동 + 무료→유료 전환 검증.
- **전술**: 공개 갤러리(M1) 오픈, 추천 프로그램 가동, 첫 창작 챌린지 개최. 페이월 A/B(9,900 vs 14,900, `09` 실험3).
- **KPI**: 무료→유료 전환율 **≥ 3%**, 결제 완료율 ≥ 60%, K-factor ≥ 0.3, **MAU 3,000 돌파**.

### 4-5. Phase 3 — 정식 런칭(GA)
- **목표**: 인지 스파이크 + 스케일.
- **런칭 채널(D-day 동시 포격)**:
  | 채널 | 용도 |
  | --- | --- |
  | **디스콰이엇(Disquiet)** | 국내 메이커·얼리어답터, 한국 GTM 1순위 |
  | **긱뉴스(GeekNews)** | 개발자·테크 커뮤니티 신뢰 |
  | **Product Hunt** | 글로벌 노출(영문 랜딩 병행) |
  | **모비인사이드·아웃스탠딩 등 미디어** | 보도자료(창작자 후기 중심) |
  | **인스타/유튜브 인플루언서** | 웹소설·글쓰기·취미 크리에이터 협찬 |
- **KPI**: 런칭일 가입 스파이크, 4주 내 **MAU 1만 진입 궤도**, LTV/CAC ≥ 3 유지.

## 5. 📰 콘텐츠·커뮤니티 마케팅

### 5-1. 공개 갤러리(M1) = SEO·바이럴 엔진
- 우수 생성작/표지를 **공개 갤러리**에 큐레이션 → 검색 유입(네이버·구글) + 자랑 동기.
- 작품 페이지마다 **고유 URL + OG 표지 이미지** → SNS 공유 시 썸네일이 곧 광고.
- "AI 웹소설 만드는 법", "○○ 장르 설정 예시" 등 **롱테일 SEO 콘텐츠**를 갤러리와 연결.

### 5-2. 우수작 큐레이션
- **주간 베스트 표지/작품** 선정 → SNS·뉴스레터·디스코드 동시 노출.
- 선정 창작자 인터뷰 콘텐츠화(B 퍼소나 락인 + C 퍼소나 신뢰).

### 5-3. 창작 챌린지·공모전
| 포맷 | 내용 | 그로스 효과 |
| --- | --- | --- |
| **주간 설정 챌린지** | "이번 주 키워드로 1화 만들기" | 생성·공유 반복(리텐션+바이럴) |
| **월간 공모전** | 테마 단편 공모, 상금/크레딧/노출 | UGC 대량 생산, 미디어 후크 |
| **콜라보 챌린지** | 인플루언서/카페와 공동 주최 | 신규 채널 유입 |
- 챌린지 출품작은 **해시태그 + 워터마크 표지**로 자동 확산 → 3장 바이럴 루프와 결합.

## 6. 🎁 리퍼럴·추천 프로그램

### 6-1. 설계 (원가 가드레일 내)
> 보상은 **현금이 아니라 크레딧**으로(원가 통제). 크레딧 1건 원가는 변동비이며, 보상 크레딧은 **신규 가입(Aha 유도)이라는 미디어 비용**으로 본다. **단, 보상 크레딧의 합산 원가 < 유료 CAC**를 항상 유지.

| 항목 | 설계 |
| --- | --- |
| **보상 구조** | 양방향: 초대자 +5크레딧 / 피초대자 +5크레딧(가입+첫 생성 시 지급) |
| **지급 조건** | 피초대자가 **Aha(1회차+표지 완성)** 도달 시에만 지급 → 허수·어뷰징 차단 |
| **상한** | 1인당 월 추천 보상 상한(예: 10명/50크레딧)으로 비용 캡 |
| **부정 방지** | 자가초대·다계정 탐지(`05` 자전거래 모니터링 로직 재사용) |
| **프리미엄 보상** | 누적 추천 N명 시 Basic 1개월·표지 고급팩 등 비현금 보상 |

### 6-2. 원가 검증 룰
- 보상 크레딧 원가는 `09` 실험2(생성 건당 실원가) 측정 후 **확정**한다. 원가>예상이면 보상 수량 하향.
- **추천 1건당 실효 비용 = 지급 크레딧 × 건당 원가**. 이 값이 동일 가입을 유료 광고로 얻는 CPL보다 낮아야 추천 프로그램을 유지한다.

## 7. 🔂 리텐션 루프

> 그로스는 "새 양동이"보다 "구멍 막기"가 먼저. 활성화(Aha) 후 **다음 회차 생성**으로 묶는 것이 핵심.

| 루프 | 장치 | 타겟 |
| --- | --- | --- |
| **이어쓰기 넛지** | "다음 화 궁금하지 않아?" 미리보기 1문단 + 1클릭 생성 CTA | A·B |
| **연재 알림** | 구독작/내 작품 후속화 푸시·카톡 알림 | C(독자)·창작자 |
| **주간 다이제스트** | 내 작품 통계(조회·후원), 추천작, 챌린지 안내 메일/카톡 | 전체 |
| **창작자 락인** | **정산 대시보드(S1)·팬덤(후원 랭킹·뱃지)** → 떠나면 잃는 자산 형성 | B |
| **연속 생성 보상** | 연속 N일 생성 시 보너스 크레딧(스트릭) | A |
| **팬덤 형성** | 후원자 보상·목표후원 게이지(`05`)로 창작자–독자 결속 | B·C |

> `08` 인사이트: 후원받은 창작자는 떠나지 않고 더 만든다 → **구독 전환·트래픽으로 간접 매출**. 리텐션 루프의 핵심 타겟은 **창작자(공급)**.

## 8. 💳 가격 주도 성장 (PLG)

### 8-1. 무료체험 → 페이월 동선
```
[가입] → [무료: 1회차 본문 + 표지 1회](Free 체험 크레딧, 05)
   → [Aha 도달] → [다음 생성 시도] → [페이월 노출]
   → [첫구매 할인] → [Basic/크레딧 충전 전환]
```
- 무료 제공은 **딱 Aha까지**(1회차+표지 1회). 가치를 체험시키되 **결제 직전에 끊는다**(`01` 플로우 원칙).
- Free는 **워터마크·판매 불가**(`05`) → 자랑/수익 욕구가 곧 전환 동기.

### 8-2. 전환 최적화 전술
| 전술 | 내용 |
| --- | --- |
| **첫구매 할인** | 페이월 첫 노출 시 한정 할인(예: Basic 첫 달 50%) + 카운트다운 |
| **앵커링** | Pro를 먼저 보여주고 Basic을 '합리적 선택'으로 배치 |
| **소진 시점 페이월** | 체험 크레딧 소진 직후 = 욕구 최고점에 결제 제안 |
| **간편결제 필수** | 카카오페이·네이버페이·토스 → 결제 완료율↑(`09` 목표 ≥60%) |
| **가격 A/B** | 9,900 vs 14,900 실측(`09` 실험3) 후 확정 |

### 8-3. 퍼소나별 PLG 경로
- **A(10대)**: 무료+소액 → 크레딧 충전(부모 결제 의존) → 워터마크 제거 욕구가 Basic 전환 트리거.
- **B(작가)**: 생산성 가치 명확 → **Pro 직행** 가능. 일괄생성·고급문체로 유료 정당화.
- **C(독자)**: 구독 거부감 적음(`01`) → 가벼운 Basic·구매로 진입.

## 9. ❄️ 콜드스타트 대응

> `00`·`05` 원칙: **마켓·후원은 작게, 도구 먼저.** 양면 시장의 닭-달걀을 피하려면 **공급(창작자)부터** 채운다.

### 9-1. 도구 우선 전략
- 런칭 초기엔 **창작 도구(구독·크레딧)** 만으로 가치 완결 → 마켓 없이도 혼자 쓸 이유 충분.
- 마켓·후원은 **콘텐츠가 충분히 쌓인 뒤**(Phase 2 갤러리 이후) 점진 오픈.

### 9-2. 초기 공급(창작자) 시딩 전술
| 전술 | 내용 |
| --- | --- |
| **시드 창작자 영입** | 작가 커뮤니티·X에서 50~100명 직접 초대, 무료 Pro 크레딧 제공 |
| **컨시어지 운영** | `09` 실험4처럼 초기 후원/판매는 수동 운영으로 사례 만들기 |
| **갤러리 시딩** | 내부·시드 창작자 작품으로 갤러리 '비어있지 않게' 선채움 |
| **표지 챌린지** | 공급자가 표지부터 만들게 유도 → 가벼운 진입으로 첫 생성 경험 |
| **인플루언서 창작자** | 웹소설 인플루언서를 '1호 창작자'로 → 그의 팬이 독자(C)로 유입 |

### 9-3. 수요(독자) 시딩
- 공급이 일정 임계(작품 N개) 넘으면 C 퍼소나 채널(인스타·카페) 본격 가동.
- 독자 유입 전까지 후원/판매 KPI를 무리하게 보지 않는다(도구 매출로 생존).

## 10. 💰 예산 배분 원칙 & 가드레일

### 10-1. 채널 예산 배분 원칙
> "**바이럴/콘텐츠로 싸게, 유료 광고로 보강**". 초기 예산은 CAC가 낮은 순서로 집중.

| 우선순위 | 영역 | 배분(초기 가이드) | 성격 |
| :-: | --- | :-: | --- |
| 1 | **커뮤니티·콘텐츠·바이럴**(디스코드/챌린지/갤러리 SEO) | ~40% | 저CAC·복리 |
| 2 | **창작자 시딩·추천 보상**(크레딧 원가) | ~25% | 공급·바이럴 |
| 3 | **유료 퍼포먼스 광고**(숏폼/메타/검색) | ~25% | 측정·스케일 |
| 4 | **런칭/PR·인플루언서** | ~10% | 스파이크 |

> 배분은 고정이 아니라 **채널별 CAC 실측으로 분기마다 재조정**. CAC 낮은 채널로 예산 이동.

### 10-2. CAC < LTV 가드레일
- **LTV(개략) = ARPPU × 유료전환율 × 평균 구독 유지 개월 ÷ 1(단순화)**. `08` 기본 시나리오 기준 보수적으로 산정.
- **가드레일: LTV / CAC ≥ 3**. 미달 채널은 축소·중단.
- **CAC 측정 = `09` 실험1 CPL(≤3,000원) → 활성화율 → 유료전환율**로 단계 환산.
- **AI 원가 가드레일 동반**: 크레딧 단가 > API 원가(`08` 함정) 위반 시, 무료 크레딧·추천 보상부터 축소(그로스가 적자를 키우지 않게).

### 10-3. 분기별 그로스 목표 (MAU 마일스톤)
| 단계 | MAU 목표 | 의미(`08`) | 핵심 과제 |
| --- | :-: | --- | --- |
| **Q1 (오픈베타)** | **3,000** | 생존 미달, 루프 검증 단계 | 활성화율·K-factor·전환율 검증 |
| **Q2~Q3 (GA)** | **10,000** | **생존선** | 바이럴+퍼포먼스 병행, LTV/CAC≥3 확립 |
| **Q4+ (스케일)** | **30,000** | **본궤도·흑자 시작** | 채널 다각화, 창작자 락인, 마켓 본격화 |

> 모든 분기 게이트: **흑자 변수 1순위 = MAU**(`00`). 단, MAU를 적자(원가>단가)로 사면 무의미 → **CAC<LTV & 크레딧단가>원가** 두 가드레일을 동시 통과해야 예산 증액.

## 11. 📐 측정 체계(요약) — `15` 연계

| 단계 | 핵심 지표 | 비고 |
| --- | --- | --- |
| 획득(Acquisition) | CPL, CAC, 채널별 가입수 | ≤3,000원(`09`) |
| 활성화(Activation) | 가입→1회차+표지 완성률 | Aha, ≥40% 목표 |
| 리텐션(Retention) | D1/D7/D30, 재생성률 | 창작자 우선 |
| 추천(Referral) | K-factor, 추천 가입 비중 | ≥0.3 |
| 매출(Revenue) | 무료→유료 전환율, ARPPU, LTV/CAC | ≥3%·≥3 |

> 상세 퍼널·지표 정의는 `15_지표체계(퍼널)` 참조. 경쟁/시장 포지셔닝 근거는 `18_경쟁시장분석` 참조.
