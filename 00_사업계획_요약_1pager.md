# 플롯위버(PlotWeaver) — 사업계획 요약 (1-Pager)

> AI 기반 웹소설 생성 + 창작자 수익화 플랫폼 / 전체 기획의 한 장 요약
> 작성: 수석 PM / 최종수정: 2026-06-26

---

## 💡 한 줄 정의
> **"설정만 넣으면 AI가 회차별 웹소설과 책 표지까지 완성해 주고, 만든 작품으로 후원·판매까지 할 수 있는 창작 스튜디오"**

---

## 🎯 문제 & 해결
| 문제(Pain) | 해결(Solution) |
| --- | --- |
| 이야기 아이디어는 있지만 문장력·끈기 부족으로 완성 못 함 | 설정 입력 → AI가 회차별 본문 자동 생성 |
| 설정이 회차 진행하며 무너짐(캐릭터 붕괴) | 관계도 + 관계변화 타임라인으로 일관성 유지 |
| 표지 등 시각 결과물 제작 어려움 | AI 표지 자동 생성 |
| 내 작품을 보여주고 응원/수익화할 곳이 없음 | 무료 공개 + 자율 후원 + 유료 판매 |

---

## 👥 타겟
- **핵심**: 소설 창작·읽기에 취미가 있는 **10~40대**
- 페르소나: ①설정덕후 입문 창작자(10대) ②연재 부담 현직 작가(20·30대) ③취미·힐링 소비 독자(30·40대)

---

## ⚙️ 핵심 기능 (MVP)
1. **설정 입력 위저드** (시대·인물·장르·목표·갈등·흐름·관계·결말·시점·회차·등급·제목)
2. **회차별 본문 자동 생성** ← 핵심 Aha
3. **AI 책 표지 생성** ← 바이럴 트리거
> 차별화: **관계도 기반 일관성 엔진** + **설정→본문→표지→판매 원스톱**

---

## 💰 수익 모델 (3대 엔진)
| 수익원 | 회사 수취 | 역할 |
| --- | :---: | --- |
| ① 구독·크레딧 | 100% | **메인 매출(생존선)** |
| ② 판매 수수료 | 30% | 창작자 수익화 + 리텐션 |
| ③ 자율 후원 수수료 | 10~20% | 진입장벽↓·트래픽·팬덤 |

> 후원/판매는 직접 매출보다 **창작자를 묶어두는 생태계 장치**. 매출의 ~90%는 구독에서 발생.

---

## 📊 매출 시뮬레이션 (월, 가정 기반)
| 시나리오 | MAU | 월 매출 | 비고 |
| --- | :---: | ---: | --- |
| 보수적 | 3,000 | 약 75만원 | 생존 미달 |
| 기본 | 10,000 | 약 608만원 | 생존선 |
| 낙관적 | 30,000 | 약 3,720만원 | 본궤도·흑자 |

> 매출 핵심 변수 = **MAU > 구독전환율 > 단가**. 흑자는 MAU 3만부터.

---

## ⚠️ 핵심 리스크 & 대응
| 리스크 | 대응 |
| --- | --- |
| **AI 원가 > 판매가 → 쓸수록 적자** | 크레딧 단가에 API 원가+마진 반영(P0 검증) |
| AI 글에 대한 지불 저항 | 무료+자율후원으로 페이월 우회 |
| 30화 일관성 붕괴(품질) | 관계도 일관성 엔진 + 품질 PoC 선검증 |
| 저작권·심의·미성년 보호 | 자동필터+사람검수 3단 방어선, 성인인증·결제한도 (`06`·`07`) |
| 마켓 콜드스타트 | 도구 먼저, 마켓·후원은 작게 시작 |

---

## 🧪 검증 계획 (4주·100만원)
1. **수요**(랜딩+가짜문) → 2. **품질·일관성**(30화 PoC) → 3. **유료 전환율**(가격 A/B) → 4. **후원·구매**(컨시어지) → *(최종)* **토큰 원가·요약방식**(모델 확정 시)
> **P0(수요·품질) 통과 전 본개발 금지.** **토큰 원가 정밀 검증은 최종 단계로 이연**(모델 확정 시점) — 가격·단가를 config로 분리해 마지막에 조정하는 전략(`09` §1 검증 순서 결정, 2026-06-29). ⚠️ 원가가 분량·생성단위 변경을 요구할 구조 리스크는 감수.
