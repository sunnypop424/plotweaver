# 필요정보(필수 입력) 정의서

> 소설 생성을 위해 **반드시** 입력받아야 하는 정보 / 모든 항목은 `Select(추천값) + 직접입력(자유 텍스트)` 하이브리드 방식

---

## 0. 입력 설계 원칙
- **Select 우선, Input 보조**: 초보자는 추천 옵션으로 빠르게, 숙련자는 직접입력으로 디테일하게.
- 모든 Select에는 마지막에 **`+ 직접입력`** 옵션을 둬서 자유 확장 허용.
- 필수 6종을 다 채워야 '생성하기' 버튼 활성화 (단, 일부는 'AI 추천 자동 채움' 제공).

---

## 1. 배경 시대 정보 (필수)
- **역할**: 작품 세계의 시간적·공간적 배경을 결정 → 본문의 어휘·소품·갈등 설정 기준.
- **입력 방식**: Select + 직접입력

| 입력 타입 | 예시 |
| --- | --- |
| Select 옵션 | 고대/중세 유럽, 동양 무협, 조선시대, 근현대, 현대 도시, 근미래 SF, 포스트 아포칼립스, 이세계/판타지 |
| 직접입력 | "2087년 화성 식민도시", "조선 후기 + 증기기관" 등 자유 서술 |

```json
{ "era": { "type": "select|custom", "value": "중세 유럽", "detail": "왕정 붕괴 직전" } }
```

---

## 2. 등장인물 정보 (필수, 1명 이상)
- **역할**: 캐릭터의 외형·성격·역할을 고정해 회차 간 묘사 일관성 유지.
- **입력 방식**: 인물 카드 단위 반복 추가. 각 필드 Select + 직접입력.

| 세부 항목 | 입력 방식 | 예시 옵션 |
| --- | --- | --- |
| 이름 | 직접입력 (+AI 자동작명) | "카엘", "이도윤" |
| 외형 정보 | Select+입력 | 흑발/금발/은발, 장신/단신, 날카로운 인상, 동안 |
| 신체적 특징 | Select+입력 | 흉터, 이종족 특징(엘프 귀), 의수, 이색 눈동자 |
| 성격 | Select+입력 | 냉철, 다혈질, 츤데레, 음흉, 순수 |
| 주인공 여부 | Select(토글) | 주인공 / 조연 / 악역 / 조력자 |

```json
{ "characters": [
  { "name": "카엘", "appearance": "은발 장신", "trait": "왼눈 흉터", "role": "주인공" }
]}
```

---

## 3. 장르 (필수, 복수 선택 가능)
- **역할**: 서사 톤·클리셰·전개 속도의 기준. 본문 생성 프롬프트의 핵심 변수.
- **입력 방식**: Select(다중 선택) + 직접입력

| 입력 타입 | 예시 |
| --- | --- |
| Select 옵션 | 회귀, 복수, 빙의, 환생, 로맨스, 로맨스판타지, 무협, 헌터/게이트, 학원물, 피카레스크, 성장물 |
| 직접입력 | "회귀+요리+힐링" 같은 혼합 장르 자유 조합 |

---

## 4. 스토리 사건의 큰 흐름 (필수)
- **역할**: 전체 서사의 뼈대(기-승-전-결). AI가 회차를 분배·전개하는 기준선.
- **입력 방식**: 단계별 텍스트 입력 + AI 추천(시놉시스 자동 초안).

| 구조 | 입력 예시 |
| --- | --- |
| 발단 | 주인공이 배신당해 죽고 10년 전으로 회귀 |
| 전개 | 회귀 지식으로 세력을 키움 |
| 위기 | 원수가 같은 회귀자임을 알게 됨 |
| 절정/결말 | 최종 대결, 복수 완성 |

> 💡 '한 줄 시놉시스만 입력' → AI가 4단 구조 자동 제안하는 옵션 제공.

---

## 5. 인물 간의 관계 (필수, 관계도 + 변화 타임라인)
- **역할**: 캐릭터 간 감정선/이해관계를 데이터화 → 회차가 진행돼도 관계 붕괴 방지. **이 서비스의 핵심 차별화 기능.**
- **입력 방식**:
  - 2명 이상의 인물을 노드로 두고 **관계(엣지)** 설정 → 관계도(그래프) 시각화.
  - 관계 라벨: Select + 직접입력.
  - **스토리 흐름에 따른 관계 변화** 설정 가능(특정 회차/단계에서 관계 전환).

| 항목 | 입력 방식 | 예시 |
| --- | --- | --- |
| 관계 라벨 | Select+입력 | 연인, 원수, 사제, 형제, 동맹, 배신자, 짝사랑 |
| 관계 방향 | Select | 일방향 / 양방향 |
| 관계 변화 | 타임라인 입력 | "5회차: 동료 → 연인", "12회차: 연인 → 원수" |

```json
{ "relations": [
  { "from": "카엘", "to": "리나",
    "timeline": [
      { "stage": "발단", "label": "동료" },
      { "stage": "위기", "label": "배신자" }
    ]}
]}
```

---

## 6. 회차(분량) 정보 (필수)
- **역할**: 생성 단위와 전체 규모 결정 → 크레딧 차감/요금과 직결.
- **입력 방식**: Select + 직접입력(숫자).

| 항목 | 입력 방식 | 예시 |
| --- | --- | --- |
| 총 회차 수 | Select+숫자입력 | 단편(1), 10화, 30화, 직접입력 |
| 회차당 분량 | Select | 짧게(2천자), 보통(4천자), 길게(6천자+) |
| 생성 단위 | Select | 1회차씩 생성 / 일괄 생성 |

---

---

## 7. 작품 제목 (필수)
- **역할**: 작품 식별·표지 텍스트·마켓 노출의 첫 접점. 판매 전환에 직결.
- **입력 방식**: 직접입력 + **AI 자동 추천**(설정 기반 제목 후보 5개 제시 → 선택).
- 미입력 시 1회차 생성 후 AI가 제목 자동 부여.

---

## 8. 주인공의 목표 · 동기 (필수)
- **역할**: 서사의 추진력. '주인공이 무엇을 원하는가'가 없으면 사건이 나열만 되고 몰입이 끊김.
- **입력 방식**: Select + 직접입력

| 입력 타입 | 예시 |
| --- | --- |
| Select 옵션 | 복수, 생존, 신분상승, 사랑 쟁취, 세계 구원, 가족 보호, 최강이 되기, 진실 규명 |
| 직접입력 | "잃어버린 동생을 찾아 대륙을 횡단한다" 등 |

```json
{ "goal": { "type": "select|custom", "value": "복수", "detail": "가문을 멸문시킨 황실에 대한 복수" } }
```

---

## 9. 핵심 갈등 (필수)
- **역할**: 이야기의 긴장을 만드는 중심 대립축. 주인공의 목표를 가로막는 장애.
- **입력 방식**: Select + 직접입력

| 입력 타입 | 예시 |
| --- | --- |
| Select 옵션 | 인물 대 인물(라이벌·악역), 인물 대 사회(체제·신분), 인물 대 운명, 인물 대 자아, 인물 대 자연/재난 |
| 직접입력 | "회귀 전 자신을 배신한 동료가 같은 회귀자" 등 |

---

## 10. 화자 시점 (POV) (필수) — *선택→필수 승격*
- **역할**: 본문 문장 생성의 기본 골격. 몰입 방식 결정.
- **입력 방식**: Select (기본값: 3인칭 전지적)

| Select 옵션 | 설명 |
| --- | --- |
| 1인칭 주인공 | "나는…" 몰입형, 웹소설에서 흔함 |
| 1인칭 관찰자 | 곁에서 지켜보는 화자 |
| 3인칭 전지적 | 모든 인물 내면 서술(기본값) |
| 3인칭 제한적 | 특정 인물 시점에 한정 |

---

## 11. 결말 방향성 (필수)
- **역할**: 전체 서사가 수렴할 지점. AI가 절정·결말 회차를 설계하는 기준.
- **입력 방식**: Select + 직접입력

| Select 옵션 | 예시 |
| --- | --- |
| 해피엔딩 / 새드엔딩 / 열린 결말 / 반전 결말 / 복수 완성 / 회귀·환생 성공 | 직접입력으로 상세 지정 가능 |

---

## 12. 연령 등급 · 수위 (필수) — *선택→필수 승격 (마켓 판매·심의 필수)*
- **역할**: 콘텐츠 노출 범위·마켓 판매 분류·심의의 기준. 판매하려면 반드시 지정.
- **입력 방식**: Select

| Select 옵션 | 설명 |
| --- | --- |
| 전체이용가 / 15세 / 19세 | 등급에 따라 표현 수위·노출 채널 제한 |

---

## ✅ 필수 입력 체크리스트 (생성 버튼 활성화 조건)
- [ ] 배경 시대 1개
- [ ] 등장인물 1명 이상(주인공 1명 포함)
- [ ] 장르 1개 이상
- [ ] 스토리 큰 흐름(최소 1줄 시놉시스)
- [ ] 인물 관계 1개 이상(인물 2명 이상일 때)
- [ ] 회차 수 · 분량
- [ ] 작품 제목(직접입력 또는 AI 자동 — 생성 후 확정 가능)
- [ ] 주인공의 목표 · 동기
- [ ] 핵심 갈등
- [ ] 화자 시점(POV)
- [ ] 결말 방향성
- [ ] 연령 등급 · 수위

> **위저드 단계는 6단계 → '기본설정(시대·인물·장르) → 서사설정(목표·갈등·흐름·결말) → 관계도 → 출력설정(시점·회차·등급·제목)'의 4그룹**으로 재구성해 입력 피로를 낮춤. 각 항목은 AI 추천값 자동 채움 지원.
