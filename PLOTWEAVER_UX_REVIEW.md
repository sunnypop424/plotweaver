> 작성 기준: 2026-06-30  
> 검토 방식: 4명의 가상 작가 페르소나가 각자의 목적으로 플랫폼을 직접 사용하는 시나리오를 시뮬레이션

## 페르소나 1 — 김지수 (25세, 입문자)

**배경:** 대학생. 웹소설 첫 도전. 판타지 로맨스 구상. AI 도구 경험 없음.  
**목표:** "내 머릿속 설정을 AI가 정리해서 1화를 써줬으면."

### 워크플로우 시뮬레이션

#### Step 1 — 내 작업실 진입
- `+ 새 작품` 버튼이 명확 ✅
- 빈 화면에 "첫 작품을 만들어보세요" 안내 ✅

#### Step 2 — 세계관 설정 (C0WorldWizard)
- 배경 시대 선택 → 장르 목록 자동 교체 ✅ (시대별 분류 키트: 무림/판타지/현대/기본)
- "✦ AI 세계관 자동 생성" → 세력·계급·지역·지도 일괄 생성 ✅
- **문제:** "국가·세력" vs "계급·지위" 탭 차이가 직관적이지 않음 — 같은 거라고 착각 ⚠️
- 월드맵 탭 — 빈 캔버스 처음 보면 "여기에 뭘 해야 하지?" 막막함 ⚠️
- 파워시스템 토글 OFF 상태에선 내용 숨겨져 있어 기능 존재를 몰랐음 ⚠️
- "건너뛰기" 버튼이 있어 필수 여부 불명확 — 나중에 필수항목 경고 뜨면 혼란 ⚠️

#### Step 3 — 인물 설정 (C1SettingsWizard)
- "✦ AI 추천 자동채움 (5명)" → 인물 카드 5개 생성 ✅
- 심리 프로필 (욕망·공포·습관·비밀) 접힘/펼침 자연스러움 ✅
- **문제:** 세계관에서 세력을 입력하지 않으면 "소속" 드롭다운이 비어서 선택 불가 ⚠️

#### Step 4 — 관계도 (C2RelationshipBuilder)
- AI 자동추천으로 관계선 자동 생성 ✅
- 그래프 시각화가 직관적 ✅
- **문제:** 회차별 관계 변화 타임라인 입력 방법을 처음엔 몰랐음 ⚠️

#### Step 5 — 서사 설정 (CNarrativeWizard)
- 힌트 텍스트 + AI 제안으로 기승전결 채우기 쉬움 ✅
- **문제:** AI 서사를 마음에 안 들어서 다시 누르면 기존 내용 덮어씀 ⚠️
- 복선 계획표·회차 패턴 등 고급 설정이 초보에게 부담 ⚠️

#### Step 6 — 출력 설정 + 생성
- 회차수·분량·시점 선택 직관적 ✅
- 필수항목 경고 배너로 빠진 항목 안내 ✅
- 생성 로더: 진행률 바 + 팁 + 입력 요약 표시 ✅
- **문제:** 생성 중 취소 버튼 없음 (최대 60초 대기) ⚠️

#### Step 7 — 에디터
- 1화 바로 뜨고 읽기 모드로 표시 ✅
- AI 검토 → 이슈별 개선안 → 개선하기 → 재검토 → 자동저장 흐름 깔끔 ✅
- **문제:** 편집 중 자동저장 없음 — 탭 닫으면 유실 ⚠️⚠️
- **문제:** 2화 생성 버튼이 사이드바 맨 아래에 있어 처음엔 못 찾음 ⚠️

## 페르소나 2 — 박민준 (32세, 중급 작가)

**배경:** 블로그 2년 연재 경험. 학원 현대물 전문. AI를 보조 도구로 활용.  
**목표:** "설정한 인물들이 AI 본문에서 캐릭터답게 행동하길. 재생성해도 일관성 유지."

### 워크플로우 시뮬레이션

#### 설정 → 집필
- 인물 심리 프로필 (욕망·공포·비밀·습관) → AI 본문 대사·행동에 실제 반영 ✅
- 관계 타임라인 설정 → AI가 회차별 관계 변화를 자연스럽게 처리 ✅
- **문제:** "말투·버릇" 입력 필드가 있는데 대사에 얼마나 반영되는지 불명확 ⚠️

#### 에디터 집중 사용
- 재생성 → AI 검토 → 개선하기 → 재검토 → 자동저장 루프 효율적 ✅
- **문제:** 재생성 버전 히스토리 없음 — 어느 버전이 더 나은지 비교 불가 ⚠️
- **문제:** 여러 회차 쓴 후 전체 흐름 일관성 검토 기능 없음 ⚠️
- **누락 플로우:** 회차별 요약·메모 없음 — 이전 화 내용 파악에 시간 소요 ⚠️

#### 설정 편집 (위저드 재진입)
- 에디터 → "설정 편집" → 위저드 편집 모드 ✅
- 스텝 헤더 클릭으로 원하는 단계 바로 이동 ✅
- 뒤로가기 → 에디터로 바로 복귀 (어느 스텝에서나) ✅
- 저장 후 에디터로 복귀 ✅
- **문제:** 설정 변경이 기존 회차에 소급 반영 안 됨 — 명시적 안내 필요 ⚠️

| 인물 설정 깊이 | ⭐⭐⭐⭐⭐ |
| AI 검토 품질 | ⭐⭐⭐⭐ |
| 편집 유연성 | ⭐⭐⭐ |
| 연속 집필 흐름 | ⭐⭐⭐ |
| **종합** | **⭐⭐⭐⭐ (3.8/5)** |

**장점:** 심리 프로필이 AI 일관성에 실질 기여. AI 검토+개선안 워크플로우 효율적.  
**단점:** 버전 비교 불가. 전체 흐름 검토 없음. 설정 변경 소급 방법 없음.

## 페르소나 3 — 이서연 (28세, 전업 작가)

**배경:** 네이버 시리즈 로맨스 판타지 연재 중. 주 3~4화 업로드. 속도가 생명.  
**목표:** "빠르게 초안 뽑고 최소 손질 후 업로드. 표지도 빠르게."

### 워크플로우 시뮬레이션

#### 기존 작품에 새 회차 추가
- 작업실 → 작품 → 에디터 바로 진입 ✅
- 사이드바 "+ 다음 회차 생성" → 이전 화 기반 연속 집필 ✅
- 생성 중 에디터 내 오버레이 + 가짜 진행 바 안내 ✅
- **문제:** 생성 중 다른 화를 편집할 수 없음 (블로킹) ⚠️

#### 자동저장 부재
- 편집 → 읽기 전환 시 자동저장 ✅
- **치명적 문제:** 편집 중 실수로 탭 닫거나 새로고침 시 유실 ⚠️⚠️⚠️
- 저장 상태 인디케이터 ("✓ 저장됨") 있음 — 하지만 자동저장 트리거가 충분하지 않음 ⚠️

#### 표지 생성 속도
- 4장 동시 생성 → 선택 → 확정 ✅
- 추가 프롬프트 (구도·자세) + 참고 이미지 업로드 (최대 3장, 640px 리사이즈) ✅
- 확정 후 작업실 카드 썸네일에 바로 반영 ✅
- **문제:** 생성에 30~60초 걸리는데 중단 방법 없음 ⚠️
- **치명적 문제:** 확정된 표지 이미지 다운로드 버튼 없음 ⚠️⚠️

#### 독자 뷰어·공유
- `/read/:id` — 배경 테마(밝게/세피아/어둡게) + 글자 크기 14~26px 설정 ✅
- 판매 페이지(`/market/:id`) — 작품 정보 표시, 미리보기 1~2화 ✅
- **문제:** 실제 결제 미구현 — 구매 버튼 클릭 시 시뮬레이션만 ⚠️

### 페르소나 3 평가

| 항목 | 점수 |
|------|------|
| 집필 속도·효율 | ⭐⭐⭐ |
| 표지 생성 | ⭐⭐⭐⭐ |
| 저장 안전성 | ⭐⭐ |
| 연재 지속 플로우 | ⭐⭐⭐ |
| **종합** | **⭐⭐⭐ (3.0/5)** |

**장점:** AI 초안 품질이 업로드 가능 수준. 표지 4장 동시 생성 후 선택 UX 좋음.  
**단점:** 자동저장 없음 치명적. 표지 다운로드 불가. 실제 결제 없음.

## 페르소나 4 — 최동현 (45세, 베테랑 작가)

**배경:** 무협·다크 판타지 전문. 세계관 복잡, 인물 20~30명. 기존에 Word/Notion 설정집 관리.  
**목표:** "복잡한 세계관과 많은 인물을 체계적으로 관리. AI가 설정을 정확히 지켜야."

### 워크플로우 시뮬레이션

#### 세계관 설정 한계
- 국가·세력 20개, 계급 체계 15개 입력 가능 ✅
- 용어사전(글로서리)으로 고유명사 관리 ✅
- 파워시스템 등급·핵심규칙·한계 상세 설정 ✅
- **문제:** 세력 간 관계(우방/적대/중립) 입력 필드 없음 ⚠️⚠️
- **문제:** 지역 지도 핀이 많아지면 겹쳐서 구분 어려움 ⚠️

#### 인물 관리 한계
- **문제:** 인물 20명+ 세로 스크롤만으로 관리 ⚠️
- **문제:** 인물 검색·필터 기능 없음 ⚠️⚠️
- **문제:** 인물 그룹핑 (세력별, 역할별) 불가 ⚠️
- 관계도 20명 이상 시 그래프 복잡도 증가 → 클릭 정확도 저하 ⚠️

#### AI 본문 설정 준수도
- 시대 배경(무협 조선시대) → 문체·어휘 반영 ✅
- 파워시스템 등급 → 전투씬 등급 용어 사용 ✅
- **문제:** 인물 많을수록 조연 비중 자동 균등화 없음 ⚠️

#### 회차·설정집 관리
- 회차 목록 확인 ✅
- **누락 플로우:** 특정 화 검색·점프 없음 ⚠️
- **누락 플로우:** 설정집 내보내기 (PDF/Word) — 버튼 있으나 showToast만 ⚠️⚠️
- **누락 플로우:** 회차별 자동 요약 없음 ⚠️

## 개선 제안 (구체적 구현 방향)

### 1. 편집기 자동저장
```
현재: 편집→읽기 전환 시, 저장 버튼 클릭 시만 저장
개선: onChange debounce 3초 자동저장 + "마지막 저장: 방금 전" 인디케이터
구현: useRef로 debounce 타이머, updateChapter(novelId, num, body) API 호출
```

### 2. 표지 다운로드
```
표지 확정 화면에 "⬇ 이미지 저장" 버튼 추가
구현: fetch(imageUrl) → blob → URL.createObjectURL → a.download = "cover.jpg"
```

### 3. 회차별 AI 요약
```
각 회차 저장 시 백그라운드로 1~2줄 요약 생성·DB 저장
에디터 좌측 사이드바에 "이전 화 요약" 접기/펼치기 패널
```

### 4. 생성 취소 기능
```
C3GeneratingLoader: AbortController 전달, "취소" 버튼 추가
백엔드: generate 엔드포인트에 스트림 중단 처리
```

### 5. 세계관 간편 / 상세 모드
```
C0WorldWizard 상단 토글: 간편 / 상세
간편 모드: 시대·장르 선택 + AI 자동생성 버튼만 (3분 설정 가능)
상세 모드: 현재 전체 탭 (세력·계급·지도·파워시스템·용어사전)
```

### 6. 재생성 버전 히스토리
```
재생성 시 이전 버전 최대 3개 보관 (로컬 state)
에디터 상단 "버전 1 / 2 / 3" 탭으로 비교·선택
선택 확정 시 updateChapter 저장
```

### 7. 단체 의상 설정 (세력 카드 통합)
```
C0WorldWizard Faction 카드에 "costume: string" 필드 추가
예: "검붉은 망토, 금장 견장, 가슴에 독수리 문장"
표지 생성 프롬프트에 "○○기사단 단복: [costume]" 자동 삽입
AI 본문 인물 묘사·배경 묘사에도 반영
```

## 총평

PlotWeaver는 **AI 보조 웹소설 창작 도구**로서의 핵심 가치를 충분히 증명합니다. 위저드 5단계 설정 → AI 초안 생성 → 에디터 수정의 메인 흐름은 매끄럽고, AI 검토+개선안 자동 적용 같은 세부 기능의 완성도도 높습니다.

**현재 강점:**
- 위저드 5단계 워크플로우 완성도 ★★★★
- AI 생성 품질 (인물 일관성, 시대고증) ★★★★
- AI 검토→개선안→자동저장 루프 ★★★★
- 표지 생성 (추가 프롬프트·참고 이미지) ★★★★

**핵심 약점 3가지 (즉시 수정 권장):**

1. **자동저장 없음** — 편집 중 탭 닫으면 유실. 프로 작가의 신뢰를 가장 크게 깎는 요소
2. **표지 다운로드 불가** — 생성해도 외부에서 쓸 수 없으면 반쪽 기능
3. **회차 삭제·내보내기 미구현** — 기본 CRUD 완결 필요

인물 10명 이하 단편·중편 작가에게는 지금도 충분히 유용 **(⭐⭐⭐⭐)**, 장기 연재·복잡한 세계관을 다루는 작가에게는 추가 개발 필요 **(⭐⭐⭐)**.
